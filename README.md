# chrome-vite-solid-ts-template

Code template for Chrome Extensions based Vite + SolidJS + TS + Tailwind CSS

![](https://skillicons.dev/icons?i=vite,solidjs,ts,tailwindcss)

## Usage

start.
```sh
pnpm i 
```

dev `HMR` via nodemon.
```sh
pnpm dev
```

output `dist` for chrome extensions.
```sh
pnpm build
```
